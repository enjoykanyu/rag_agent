# 周杰伦《太阳之子》专辑

## 专辑简介

《太阳之子》是周杰伦的第16张个人专辑，于2026年3月25日全球数字发行，由杰威尔音乐与环球音乐联合出品。距离上一张专辑《最伟大的作品》（2022年7月）发行已过去三年八个月，是周杰伦出道以来间隔时间最长的专辑之一。

## 专辑名称由来

"太阳之子"这个名称源自张学友对周杰伦的认证。2023年5月，周杰伦在香港举行"嘉年华"巡演，演出前几天持续下雨，但每次几乎开演前雨就停了。张学友特地送来印有"太阳之子"字样的芒果和手写卡片，认证周杰伦是"太阳之子"。周杰伦当时说"心里出太阳了"，打趣说这个称号他拿走了。三年后，他将这个称号用来命名第16张专辑。

## 曲目列表

专辑共收录13首曲目，其中12首为全新创作歌曲，另特别收录送给歌迷的《圣诞星》：

1. 太阳之子（同名主打）
2. 淘金小镇
3. 西西里
4. 那天下雨了
5. 圣诞星（特别收录）

（其余曲目信息仍在陆续公开中）

## 口碑评价

歌迷对13首歌曲的评价呈明显梯队分化：

- **第一梯队**：《淘金小镇》《西西里》《那天下雨了》稳居好评前列，被歌迷评价为"夯爆"
- **争议焦点**：同名主打歌《太阳之子》意外垫底，引发歌迷热议

## 主打歌《太阳之子》MV

《太阳之子》MV是华语乐坛史上制作规模最大的MV之一：

- **制作费用**：接近1亿台币（约2000万人民币），创下华语乐坛MV制作费新纪录
- **制作团队**：携手新西兰维塔工作室（Wētā Workshop），该团队曾参与《阿凡达》《魔戒》《猩球崛起》《沙丘》等180多部电影的视觉特效制作，五次获得奥斯卡金像奖
- **制作周期**：从2023年12月开始策划，2025年1月在巴黎教堂开拍，2025年4月在台北搭景补拍，2026年3月完成后期，历时两年三个月
- **MV时长**：6分58秒
- **视觉亮点**：MV中融入了30幅世界名画，包括克林姆《吻》等经典作品，画中人物全部由真人还原演出，化妆、美术、道具、灯光完全按照原画复原
- **拍摄地点**：法国巴黎圣女贞德小教堂、中国台湾省台北市

## 专辑主题

周杰伦在发布会上阐释了专辑的核心主题："大家内心可能有阴暗面，你是选择消灭它，还是与它共存？"他坦言年轻时开演唱会什么都不怕，现在反而会想很多，因此希望新专辑能给大家带来阳光和正能量的感觉。《太阳之子》的主题是以光驱散内心阴暗，歌曲原计划命名为"我就是光"。

## 幕后花絮

- 周杰伦在发布会上回应了歌迷长期吐槽MV中"一直走路"的问题，表示走路是一种情绪表达，场景很美要记录下来，但歌词并没有对应的画面，所以无法对嘴。他开玩笑说新专辑13首歌，主打歌预算都用完了，剩下12首都是"走路"
- 周杰伦宣布将为小女儿Jacinda写一首歌，因为女儿看到姐姐有《前世情人》、哥哥有《粉红海洋》，已经会问"那我呢"
- 周杰伦透露将在一首MV中首次公开当年在英国古堡向昆凌求婚的完整影片
