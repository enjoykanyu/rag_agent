# 暗影蜘蛛侠第一集剧情

## 基本信息

《暗影蜘蛛侠》（Spider-Noir）是一部黑色电影风格的超级英雄剧集，由尼古拉斯·凯奇饰演主角本·莱利（Ben Reilly）。故事设定在1933年的纽约，以黑色侦探片的叙事方式重新演绎蜘蛛侠的故事。第一集标题为"Step Into My Office"，时长47分钟。

## 故事背景

在这个宇宙中，蜘蛛侠被称为"蜘蛛"（The Spider）。本·莱利获得超能力的经历与主流宇宙不同——"能力越大，责任越大"这句经典名言并非来自本叔叔，而是来自他的女友露比（Ruby）。本原本计划向露比求婚，但露比不幸溺水身亡，尽管蜘蛛拼尽全力也未能救回她。此后五年，本放弃了义警生涯，成为一名落魄的私家侦探。

## 第一集剧情

### 追踪任务

本·莱利以私家侦探的身份接到了一个名叫温斯顿（Winston）的男人的委托，要求他追踪一个叫艾迪生（Addison）的人。与此同时，另一名私家侦探唐尼格尔（Donegal）也受雇于温斯顿追踪同一个目标。

### 工厂对峙

唐尼格尔在一家石油公司的工厂里找到了艾迪生，但震惊地发现艾迪生拥有火焰超能力。唐尼格尔开枪击杀了艾迪生，但艾迪生的火焰引燃了石油，引发了剧烈爆炸。本及时赶到，救了竞争对手唐尼格尔的性命。两人决定联手调查温斯顿的真实身份。

### 黑帮阴谋

本和唐尼格尔调查后发现，雇佣他们的温斯顿实际上是为黑帮老大银貂（Silvermane）工作的。唐尼格尔想借机敲诈温斯顿要更多钱，本则坚决反对。与此同时，艾迪生此前在某人指使下烧毁了银貂的豪宅，这一事件可能促使银貂雇佣沙人（Flint Marko/Sandman）、兆瓦特（Dirk Leydon/Megawatt）和墓碑（Lonnie Lincoln/Tombstone）等超能力者作为手下。

### 罗伯森的劝说

本的好友罗比·罗伯森（Robbie Robertson）是本宇宙中的记者角色，他知道本就是"蜘蛛"。罗比试图说服本重新穿上战衣回归义警生涯，但本拒绝了，展现出比传统蜘蛛侠更加愤世嫉俗的一面。两人一同前往停尸房检查艾迪生的遗体。本请求罗比不要报道艾迪生的死讯，并试图联系唐尼格尔会面。

### 猫哈迪的委托

一个名叫卡迈迪（Carmedy）的男人雇佣本调查他妻子猫·哈迪（Cat Hardy）是否出轨。本跟踪猫到一家酒店，发现她正与市长候选人莫里斯（Morris）纠缠。本用蛛丝将莫里斯缠住，让猫得以脱身，并在干预前拍下了两人会面的照片。

### 勒索与冲突

本对猫的丈夫撒了谎，但决定用照片和底片勒索猫。当他去猫的俱乐部勒索时，猫叫来了她的保镖——弗林特·马科（Flint Marko，即沙人）将本扔了出去。与此同时，银貂绑架了唐尼格尔，得知是唐尼格尔击杀了艾迪生。银貂杀死了唐尼格尔，命令温斯顿把本带过来。

### 天台激战

弗林特跟踪本到一栋废弃建筑，本早已察觉马科在身后。马科想要底片，因为本已经在俱乐部把照片交给了猫。马科将本从楼上扔下，但本用蛛丝荡回楼上，与正在使用沙子能力的马科展开搏斗。猫出面制止了战斗，本将底片交给了她。

### 结局

第二天，本的助手珍妮特（Janet）因长期未领到工资而愤怒辞职。就在此时，猫走进本的办公室，雇佣他去寻找失踪的弗林特·马科——第一集到此结束，为后续剧情埋下伏笔。

## 角色介绍

- **本·莱利/蜘蛛**（尼古拉斯·凯奇 饰）：落魄的私家侦探，曾是义警"蜘蛛"，因女友露比之死而放弃英雄生涯
- **猫·哈迪**：神秘女性，与黑帮和政界都有牵连
- **弗林特·马科/沙人**：猫的保镖，拥有操控沙子的超能力
- **银貂**：纽约黑帮老大，幕后操控者
- **罗比·罗伯森**：记者，知道本的真实身份
- **唐尼格尔**：私家侦探，本的同行为竞争对手
- **温斯顿**：银貂的手下，负责雇佣侦探
