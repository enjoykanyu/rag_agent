# 水浒传主要人物

《水浒传》是中国古典四大名著之一，由施耐庵所著，讲述了北宋末年以宋江为首的一百零八位好汉在梁山泊聚义、替天行道的故事。以下介绍其中最具代表性的主要人物。

## 天罡星三十六将（部分）

### 宋江（天魁星·呼保义）

梁山泊第一把交椅，原为郓城县押司，绰号"及时雨"。为人仗义疏财，深得江湖好汉敬重。因怒杀阎婆惜被发配江州，浔阳楼题反诗被判死罪，被梁山好汉劫法场救出。主张招安，带领梁山好汉归顺朝廷，最终被奸臣用毒酒害死。

### 卢俊义（天罡星·玉麒麟）

梁山泊第二把交椅，原为北京大名府富商，武艺高强，棍棒天下无双。被吴用设计骗上梁山，活捉史文恭，征方腊后被奸臣害死。

### 吴用（天机星·智多星）

梁山泊军师，原为乡村教书先生，足智多谋。从智取生辰纲开始辅佐晁盖、宋江，是梁山的智囊核心，策划了多次重大行动。宋江死后自缢于其墓前。

### 公孙胜（天闲星·入云龙）

梁山泊法师，道术高深，能呼风唤雨。参与智取生辰纲后上梁山，后以探母为由离开，征方腊前归隐山林。

### 林冲（天雄星·豹子头）

原为东京八十万禁军教头，武艺超群。被高俅陷害误入白虎堂，刺配沧州，风雪山神庙手刃仇人后逼上梁山。是梁山元老级人物，屡立战功，征方腊后病逝。

### 鲁智深（天孤星·花和尚）

原名鲁达，原为渭州经略府提辖。因拳打镇关西救金翠莲而遁入空门，在五台山出家。为人嫉恶如仇、豪爽率真，倒拔垂杨柳、大闹野猪林，是水浒中最具侠义精神的人物。征方腊后在六和寺坐化圆寂。

### 武松（天伤星·行者）

绰号"行者"，景阳冈打虎闻名天下。为兄报仇杀潘金莲、西门庆，血溅鸳鸯楼，醉打蒋门神，大闹飞云浦。武艺高强，性格刚烈，是水浒中最受欢迎的人物之一。征方腊后断臂，在六和寺出家，八十岁善终。

### 关胜（天勇星·大刀）

关羽后裔，使青龙偃月刀，有乃祖之风。原为蒲东巡检，被呼延灼推荐征讨梁山，后归降。征方腊后落马病死。

### 秦明（天猛星·霹雳火）

性如烈火，使一条狼牙棒。被宋江设计骗上梁山，全家被慕容知府所杀。征方腊时战死。

### 花荣（天英星·小李广）

神箭手，箭法无双，与宋江义结金兰。梁山马军八骠骑兼先锋使之首，征方腊后任应天府兵马都统制，闻宋江死讯后在其墓前自缢。

### 柴进（天贵星·小旋风）

后周世宗柴荣嫡派子孙，家中有丹书铁券。仗义疏财，广纳天下好汉，林冲、武松等都曾受其庇护。

### 李应（天富星·扑天雕）

原为李家庄庄主，善使飞刀。被宋江设计逼上梁山，主管梁山钱粮。

### 朱仝（天满星·美髯公）

原为郓城县马兵都头，面如重枣，美髯过腹，义释宋江、雷横。被李逵劈死小衙内逼上梁山。

### 鲁智深之外的其他重要好汉

### 杨志（天暗星·青面兽）

杨家将后人，因失陷花石纲丢官，卖刀杀牛二，押送生辰纲又被劫，走投无路上二龙山落草。

### 徐宁（天祐星·金枪手）

金枪法、钩镰枪法天下无双，被吴用设计骗上梁山，教梁山军钩镰枪法破连环马。

### 索超（天空星·先锋使）

性急如火，使一把金蘸斧，与大名府杨志比武闻名。

## 地煞星七十二将（部分代表人物）

### 燕青（天巧星·浪子）

卢俊义家仆，多才多艺，相貌俊美，武艺高强。摔跤胜任原，月夜遇道君，是水浒中最灵巧的人物。征方腊后功成身退，辞别卢俊义归隐。

### 李逵（天杀星·黑旋风）

绰号"黑旋风""铁牛"，性格鲁莽暴烈，使两把板斧。对宋江忠心耿耿，江州劫法场、杀四虎、斧劈罗真人，是水浒中最具爆发力的人物。宋江饮毒酒后怕李逵造反毁其名声，也让李逵饮毒酒同死。

### 张顺（天损星·浪里白条）

水中好汉，水性极佳，在水中如履平地。涌金门战死，是征方腊中牺牲的最令人痛惜的好汉之一。

### 时迁（地贼星·鼓上蚤）

梁山谍报人员，轻功了得，擅长沙中飞檐走壁。盗甲偷鸡，多次立下奇功。

### 孙二娘（地壮星·母夜叉）

十字坡开黑店，卖人肉包子，武艺不俗，与丈夫张青一起上梁山。

### 顾大嫂（地阴星·母大虫）

登州兵马提辖孙立弟媳，为救解珍解宝劫牢，是梁山三位女将之一。

## 其他重要人物

### 晁盖（托塔天王）

梁山泊前任寨主，原为东溪村保正。智取生辰纲后上梁山，火并王伦后成为寨主。曾头市中箭身亡，临终遗言："谁捉住史文恭，谁就做梁山泊主。"

### 高俅

太尉，水浒中的大反派。原为市井无赖，因踢得一脚好球得端王（宋徽宗）赏识，发迹后迫害林冲等忠良。

### 西门庆

阳谷县恶霸，与潘金莲私通，毒杀武大郎，被武松所杀。
