# 三国演义

《三国演义》是中国古典四大名著之一，由元末明初小说家罗贯中所著，全名为《三国志通俗演义》，是中国第一部长篇章回体历史演义小说。

## 剧情概述

故事以东汉末年为背景，讲述了从黄巾起义到西晋统一近百年间的历史风云。全书可大致分为三个阶段：

### 东汉末年与群雄割据

东汉末年，朝政腐败，宦官专权，民不聊生。张角发动黄巾起义，天下大乱。在此背景下，刘备、关羽、张飞桃园三结义，立志匡扶汉室。各路诸侯纷纷起兵，董卓入京专权，被王允设计诛杀后，天下进入群雄割据的局面。曹操"挟天子以令诸侯"，逐步统一北方；孙策渡江奠定江东基业；刘备颠沛流离，寄人篱下。

### 三国鼎立

赤壁之战是全书最重要的转折点。曹操率八十万大军南下，刘备与孙权联合抗曹，诸葛亮舌战群儒促成孙刘联盟。周瑜指挥赤壁之战，火烧曹军战船，曹操大败北归。此后三国格局基本形成：曹操占据北方，刘备据有荆益，孙权坐镇江东。刘备在诸葛亮辅佐下攻取益州、汉中，称汉中王；曹操病逝，曹丕篡汉称帝，建立魏国；刘备称帝延续汉统，史称蜀汉；孙权称帝，建立吴国，三国正式鼎立。

### 三国归晋

关羽大意失荆州、败走麦城被杀，刘备兴兵伐吴，在夷陵之战中被陆逊火烧连营，大败而归，病逝白帝城，托孤于诸葛亮。诸葛亮六出祁山北伐中原，最终病逝五丈原。姜维继承遗志继续北伐，但蜀汉国力日衰。司马氏篡魏建立晋朝，先灭蜀汉，再灭东吴，天下归一，三国时代落幕。

## 主要人物

### 蜀汉阵营

- **刘备**：字玄德，汉室宗亲，仁德之君。以仁义著称，从织席贩履之辈成长为蜀汉开国皇帝，三顾茅庐请诸葛亮出山，白帝城托孤。
- **关羽**：字云长，刘备义弟，忠义无双。温酒斩华雄、过五关斩六将、水淹七军，威震华夏，后败走麦城被杀，被后世尊为"武圣"。
- **张飞**：字翼德，刘备义弟，勇猛刚烈。长坂桥一声怒吼吓退曹军百万，义释严颜，后被部下暗杀。
- **诸葛亮**：字孔明，号卧龙，蜀汉丞相。智慧超群，忠心耿耿，草船借箭、空城计、七擒孟获、六出祁山，是智慧的化身，"鞠躬尽瘁，死而后已"。
- **赵云**：字子龙，常山赵子龙。长坂坡单骑救主，七进七出曹营，忠勇兼备，是蜀汉五虎上将之一。
- **马超**：字孟起，西凉猛将，人称"锦马超"，杀得曹操割须弃袍。
- **黄忠**：字汉升，老当益壮，定军山斩夏侯渊。
- **姜维**：字伯约，诸葛亮传人，九伐中原，蜀汉最后的支柱。

### 曹魏阵营

- **曹操**：字孟德，魏国奠基者。雄才大略，"治世之能臣，乱世之奸雄"，统一北方，文武双全，也是著名诗人。
- **司马懿**：字仲达，魏国权臣。隐忍深沉，与诸葛亮多次交锋，最终发动高平陵之变，为司马氏篡魏奠定基础。
- **夏侯惇**：字元让，曹操族弟，拔矢啖睛，忠勇刚烈。
- **张辽**：字文远，合肥之战威震逍遥津，令东吴小儿止啼。
- **许褚**：字仲康，曹操贴身护卫，人称"虎痴"，裸衣斗马超。
- **郭嘉**：字奉孝，曹操首席谋士，英年早逝，被曹操视为奇才。

### 东吴阵营

- **孙权**：字仲谋，东吴开国皇帝。"生子当如孙仲谋"，善于用人，据守江东，与魏蜀三分天下。
- **周瑜**：字公瑾，东吴大都督。风流儒雅，赤壁之战火烧曹营，可惜英年早逝。
- **鲁肃**：字子敬，东吴名臣。忠厚长者，力主孙刘联盟，单刀赴会。
- **吕蒙**：字子明，"士别三日当刮目相待"，白衣渡江袭取荆州。
- **陆逊**：字伯言，夷陵之战火烧连营大败刘备，东吴柱石。
- **甘宁**：字兴霸，百骑劫魏营，东吴猛将。

### 其他重要人物

- **吕布**：字奉先，天下第一猛将，"人中吕布，马中赤兔"，但反复无常，最终白门楼殒命。
- **董卓**：字仲颖，专权乱政，祸乱朝纲，被王允设计诛杀。
- **袁绍**：字本初，四世三公，官渡之战败于曹操。
